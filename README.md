# Telegram AI-like Daily Prompt Bot 🤖

This project sends a daily randomized journal/motivation prompt to Telegram using GitHub Actions.

## Features
- 100% free (no API cost)
- Smart category rotation
- Random prompts
- Daily automation (6 AM IST)

## Setup
1. Add TELEGRAM_BOT_TOKEN in GitHub Secrets
2. Add TELEGRAM_CHAT_ID in GitHub Secrets
3. Enable Actions

Done 🚀
